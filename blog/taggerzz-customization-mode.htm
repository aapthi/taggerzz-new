CONTENT CUSTOMIZATION / CUSTOMIZATION MODE (Highlights posted by Member, Public Databoxes & Private Databoxes)

Once a collection of web links has been created, the user can go into Customization Mode at any time to make further changes. The user can reframe, resize, relocate and regroup web link posts or add comments to either a group of link posts or a single link post. All independent types of valid web links are converted to link posts upon exiting customization mode.

Displaying Link Posts as:
  • Direct web links to images will be represented by a hyperlinked thumbnail of the image
  • Web links to web pages will be represented by the web page's title, image & available content
  • Direct web links to : .doc, .pdf, .psd, .ai, .cdr, .eps, .zip, .torrent, .css, .txt files will be represented by their respective Application Icons
  • Web links to videos from major video portals will show up as thumbnail videos with overlaid playback controls

Add comment

The user can add his comments under any link post or group of link posts (For this, the user will have to create a group of individual link posts).

Refresh Data

Sometimes, the data at the location to which a link is referring to could change with time. Taggerzz.com accommodates these changes by providing the user an option to refresh the data at any time. Whenever the user wants to update his links with new data, all he/she has to do is click on “Refresh Data”. This would take the user to the link insertion page, where he/she can “continue customizing” the links there.

Share

Users can share databoxes or highlights on social networks or by email.
