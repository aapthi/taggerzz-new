Creating, Sharing, Posting, #tagging a Public Databox

START #TAGGING (From Header)

Registered users can start creating their own collection of web links by clicking on “Start #tagging” on the page header. The user can also choose to create and/or publish a highlight or a public or private databox.

Creating: Public Databoxes (Step/Page 1)

-Creating your Public Databox

The user can create a public databox on taggerzz.com in a few simple steps. The user has to first specify a #tag and then a page title. All the web links that will be a part of this Public Databox will be added next.

-#Hashtag

The user can type in his/her chosen #tag in the #tag field

-Title

The user can type in his/her chosen title for the public databox.

-Upload Bookmarks (Step/Page 2)

The user can choose to upload a bookmark file here. Taggerzz.com accepts bookmark files that have a .json or .bak file extension. Bookmark files with a .json extension are usually generated by Mozilla Firefox browsers, whereas the Google Chrome browser usually generates its bookmark files with a .bak extension.

-Browse

By clicking on ‘Browse’, the user can search for and select the bookmark file from his/her local storage.

-Upload file

Once the user selects the bookmark file, he/she needs to upload it by clicking the ‘Upload’ button. Upon successfully uploading a valid bookmark file, taggerzz.com displays all the web links from the bookmark file in their original folder structure and with the page titles that the user had originally saved them with. This makes it easy for the user to then select only the web links he/she wants to include in the current Public Databox. After the selection, all the selected web links are displayed in editable fields.

- Continue anyway

Clicking on “Continue anyway” allows the user to skip uploading a bookmark file, and create the Public Databox by manually pasting or typing in the web links he/she wants to include in the public databox.

-Creating your Public Databox (Final Step/Page)

-Hashtag

The #tag field displays the #tag that the user selected in step 1. The user can choose to make changes to this field.

-Title

The Title field displays the title that the user selected in step 1. The user can choose to make changes to this field.

-00 Hot Links

This is where taggerzz.com displays to the user the number of valid web links available in the public databox. A placeholder text that says “Articles, news items etc” hints to the user that he/she can type in something that would describe what type of public databox is being created.

-Image

The user can upload an image that would appropriately represent what the public databox is all about. Taggerzz.com facilitates cropping of the uploaded image so as to make it make it a perfect fit.

-Upload Image

By clicking on “Upload Image”, the user can select and upload an image, and crop it appropriately. This image will then be displayed on the public databox.

- The nature of your content (ie. Matured Content / NSFW Content)

Taggerzz.com allows the user to specify the nature of the web links that will be posted as a public databox. If the user feels that the web links lead to mature content, he/she can label it as such by checking the appropriate options. By doing this, he can advise other users to exercise their discretion while they access the contents of the public databox.

-Categorize Your Public Databox

Taggerzz.com encourages the user to specify appropriate keywords to categorize the web links that will be posted in the public databox. This makes the contents of the public databox easily accessible to search engines.

-Continue Customizing

After adding web links either manually or from a bookmark file, the user can continue customizing his posted web links, which will be displayed as link posts. Up to 18 major file formats can be presented as link posts.
