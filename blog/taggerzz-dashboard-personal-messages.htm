TAGGERZZ.COM'S USER DASHBOARD

The dashboard provides a convenient location for the user to perform the most common tasks on taggerzz.com. Here, he/she can access and edit all creation and sharing activities. Public and private databoxes can also be accessed and edited right from the dashboard. The following are the main features that can be accessed from the dashboard:

Public Databoxes
Highlights
Private Databoxes

Personal Messages

If the user has enabled the personal messaging service in Account Settings, he/she can send and receive messages from other registered users on taggerzz.com. When new personal messages are received, the dashboard notifies the user by displaying a counter next to Dashboard on the main menu, with the counter displaying the number of new messages that have arrived. The user can always choose to turn off personal messaging from the Account Settings.
