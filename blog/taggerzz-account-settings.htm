TAGGERZZ.COM'S USER ACCOUNT SETTINGS

Member Details

Edit/Upload your Picture

The profile picture that the user uploads to his/her account on taggerzz.com is displayed on the profile page as well as in comments posted by the user. It is also displayed as a thumbnail on the user activity summary on the page header.

Profile Name/Alias

The user can enter a user name of his/her choosing. This username can be edited at any time. Other members will identify the user by the user name he/she enters here.

Password

This is where a user types in a password. The password is requested of the user during login and any other situation where the user’s identity needs to be verified.

Email ID

This is where the user puts in his/her email id. It is important to note that taggerzz.com uses this email id to communicate with the user.

Email ID Verified

Once the user saves his email id and password, a verification mail is sent to the email id provided by the user. The user must click on the link in the verification email, to verify his email account with taggerzz.com. Only after the email id is successfully verified can the user become a registered member on taggerz.com.

Deactivate Account

If the user chooses to deactivate his/her account on taggerzz.com, then his/her creation and sharing activities are temporarily deactivated.

Hide me (in Members)

By checking this box in Account Settings, the user can rest assured that his/her profile information will not be displayed to other members on taggerzz.com.

Disable “Personal messaging”

By checking this box in Account Settings, the user can choose not to use the personal messaging service on taggerzz.com. He/she will not be able to send or receive personal messages to or from other members.

Your Activity (Account settings)

This section displays the activity of the user on taggerzz.com. This includes the total number of web links that the user has shared, the number of private and public databoxes and highlights that have been created by the user and also the number of upvotes and downvotes each one received. The full list of data displayed is as follows:

The number of user created databoxes
The number of user created highlights
The number of user created private databoxes
Montage status (tacked or un-tacked)
The total number of keywords
Responses received on posts
The total number of links collected, categorized or #tagged
