TAGGERZZ.com's Home Content

Highlights #tagged by Members

Highlights posted by registered members of taggerzz.com form the main body of content on the home page. Users can navigate the page to access newly created and shared highlights as well as top voted highlights.

Public Databoxes

Public databoxes created and shared by users are displayed below the highlights. Databoxes marked as NSFW/Mature material are only visible to logged in users. If the user searches for a #tag and a public databox with the matching #tag exists, then that databox will be displayed in the search results.

Private Databoxes

Private databoxes are never displayed alongside public databoxes. Private databoxes can be accessed only if the user searches for it by its #tag and freshly generated or user specified Unique Combination Code.

Members

Registered members of taggerzz.com can view their profiles as well as details of their activities on taggerzz.com.
