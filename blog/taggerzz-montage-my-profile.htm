MONTAGE (MY PROFILE)

The ‘Montage’ or ‘My Profile’ page is meant to display the user’s profile information. Until the user chooses to set a #tag, title and description for his/her profile, this section will be marked as un-tacked or un-set. Here, a user can display content like highlights that he/she has created or posted, public databoxes, and also private databoxes of the logged in user(to the Private Databox creator only). This page is meant to display all user activity related information in one place. If the user doesn’t want his/her member profile to be listed in the “Members” section, he/she can specify the same in the Account Settings page.

Member’s Display Picture

Here, the user can choose a profile image to represent him/her on taggerzz.com.

Member Profile’s #Hashtag

Here, the user can set a #tag to better represent his profile.

Member Profile’s Title

Here, the user can set an appropriate title that would represent all the web collections he has created and shared on taggerzz.com.

Description section

Here, the user can add a description to better explain the characteristics of the web links in the collection. This can make it easier for other members to understand the collection better.
